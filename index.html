☁️ GitHub Gist Sync

One-time setup — playlists auto-save to your private Gist after every change.

  1. Go to github.com → Avatar → Settings → Developer settings
  2. Click Personal access tokens → Tokens (classic) → Generate new token
  3. Check only the gist scope → copy the token
  4. Go to gist.github.com → New secret Gist
  5. Filename: musicpro.json · Content: {} · Create
  6. Copy the Gist ID from the URL (the long hash)
